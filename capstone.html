Quadcopter Propeller Optimisation Using CFD

Overview

This capstone project focused on enhancing quadcopter propeller efficiency using Computational Fluid Dynamics (CFD) with Multiple Reference Frame (MRF) analysis.

Project Summary

  • Analysed aerodynamic performance of baseline and redesigned blades using CFD.
  • Evaluated thrust, torque, and efficiency using MRF simulation techniques.
  • Redesigned blade geometries in SolidWorks and validated improvements in power-to-thrust ratio.
